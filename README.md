# Mykrobe Atlas

[Web](docs/web.md) and [Desktop](docs/desktop.md) client for Mykrobe Atlas.

## Install

* Install package manager

	```
	$ brew install yarn
	```

* If using `nvm`, set the Node version

	```
	$ nvm use
	```

* Install dependencies

	```
	$ yarn
	```
